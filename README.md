## Repository overview

This repository groups five related language channels into one structured access layer:

- Kinyarwanda
- Kirundi
- Luganda
- Lingala
- Kikuyu

Its purpose is clear: give this language cluster a cleaner and more useful place for discovery, integration, and multilingual Islamic content workflows.

---

## Why this repository exists

Some repositories need a broad continental label.

This one does not.

This cluster is regionally coherent enough to justify its own focused layer for:

- regional discoverability
- structured Islamic content access
- multilingual publishing workflows
- localization and deployment
- educational product building
- cleaner repository segmentation

---

## Why this cluster works

This is a practical regional cluster.

Kinyarwanda, Kirundi, and Luganda form a strong Great Lakes and nearby East African core, while Lingala and Kikuyu extend the repository into adjacent central and eastern language environments without breaking the regional logic.
